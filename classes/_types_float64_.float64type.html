  • lte(left: number, right: number): boolean

    Compares two valid float64 values.

    The values are ordered as follow:

    • -Infinity
    • Negative non-zero finite values
    • -0
    • +0
    • Positive non-zero finite values
    • +Infinity
    • NaN
